# FileCrypt

Encrypt & Decrypt a file in "AES-256-GCM" mode.

## Usage

<pre>
-encrypt [filename] [key]
        filename: no limit
        key: 32 characters
-decrypt [filename] [key]
        filename: end with ".enc"
        key: 32 characters
</pre>
